AckAgent PAM

This installer will install the AckAgent PAM module and administration CLI:

  • pam_ackagent.so - PAM authentication module
  • ackagent-pam - PAM module administration and testing tool

Installation Locations

  • /usr/local/lib/pam/pam_ackagent.so
  • /usr/local/bin/ackagent-pam

After Installation

Use ackagent-pam to configure and test PAM authentication:

  • ackagent-pam test <username> - Test authentication
  • ackagent-pam config - Show configuration
Note: To enable the PAM module, add it to your PAM configuration (e.g., /etc/pam.d/sudo). See the documentation for configuration details.
